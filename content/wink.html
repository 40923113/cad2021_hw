Wink

Google尋找 debugmode wink

打開 winksetup，點選 I Agree

按 Browse 將資料存在 date ，再按Install

在隨身碟中找尋Data/Wink資料夾/打開Wink，就可以開始製作影片了

安裝影片

到老師的Wink影片複製原始碼143-152行

貼在Source code

開啟影片原始碼 複製11-26行 貼在Source code

修改內容

在<div class="winkVideoContainerClass">後面加上<video width="960" height="540" (影片的尺寸)

data-dirname="/static"</video>

<source src="/downloads/wink 4.mp4"

將影片放進 downloads 裡 複製MP4檔

貼上到data/tmp/cad2021_hw/download裡

將複製下來的原始碼貼在 Source code

並且修改

<video width="960" height="540" (影片的尺寸)

data-dirname="/static" data-varname="winkVideoData" video="">

<source src="/downloads/wink 4.mp4"
